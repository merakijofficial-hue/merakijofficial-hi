How to upload to Vercel:
1. Unzip this folder.
2. Go to Vercel.
3. Create a new project.
4. Upload/import this folder.
5. Framework preset: Other.
6. Build command: leave blank.
7. Output directory: leave blank or use root.
8. Deploy.

After deployment, connect merakij.com in Vercel Project Settings → Domains.
